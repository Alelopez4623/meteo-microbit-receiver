## Receptor 
Este se encarga de recibir los datos de la micro:bit emisora y pasarlos mediante serial a el PC

## Edita este proyecto

Para editar este repositorio en MakeCode.

* abra [https://makecode.microbit.org/](https://makecode.microbit.org/)
* haga clic en **Import** luego haga clic en **Import URL**
* pegue **https://github.com/alelopez4623/meteo-microbit-receiver** y haga clic en importar
